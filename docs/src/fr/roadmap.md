# Feuille de route

> **En cours de traduction.** La feuille de route complète est pour
> l'instant [disponible en anglais](../roadmap.html). Sa traduction
> FR est suivie dans **F25** — la tâche même qui pilote la mise en
> place de cette version française.

## Repères

- **v0.1** livrée — 17 règles déterministes, bilingues EN/FR.
- **v0.2** en cours — 25 règles au total, modèle de score hybride
  (score global + cinq sous-scores par catégorie), polissage docs,
  miroir FR (en cours).
- **v0.3** planifiée — extensions LLM/NLP optionnelles, SARIF pour
  GitHub Code Scanning.

La version anglaise reste la référence en attendant la traduction
complète.
