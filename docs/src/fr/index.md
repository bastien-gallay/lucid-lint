# Version française — en chantier

Merci d'être passé.

<!-- lucid-lint disable-next-line unexplained-abbreviation passive-voice -->

La version française arrive en **v0.2**. `lucid-lint` parle les deux
langues depuis le début. Sa documentation doit faire pareil.

## Pourquoi pas maintenant ?

Le projet vise un lectorat aux besoins précis : troubles de
l'attention, dyslexie, fatigue cognitive, français langue seconde.
Une documentation qui sert ces lecteurs mérite mieux qu'une traduction
automatique.

<!-- lucid-lint disable-next-line unexplained-abbreviation -->

Les règles de l'outil couvrent les deux langues. Le corpus
francophone — FALC (Facile À Lire et à Comprendre), RGAA (Référentiel
Général d'Amélioration de l'Accessibilité) — est au cœur du projet.

## Pour suivre l'avancement

- Le chantier est suivi sous [**F25** dans le
  roadmap](../roadmap.html#v02---first-major-iteration).
- Les relectures et contributions en français sont les bienvenues.
  Ouvrez une
  [issue sur GitHub](https://github.com/bastien-gallay/lucid-lint/issues/new)
  avec le label `docs-fr`.
